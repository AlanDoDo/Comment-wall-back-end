## 后端

这里记录后端的内容

## 安装
```
npm install
```

## 运行
```
nodemon index.js
```

### 下载express插件
```
cnpm install express --save
```

### 创建后端目录
* |-- serve
    * |-- index.js
    * |-- package.json
    * |-- 后端开发记录.md
    * |-- config/
    * |   |-- default.js
    * |-- controller
    * |-- lib/
    * |   |-- db.js
    * |-- routes/

### 下载mysql插件
```
cnpm install mysql --save
```

### 下载multer插件

cnpm install multer --save
